# socket test
the test if it is able to communicate each other raspi using socket.

#How to test socket.
1. $ make
 compile file in each machine tested
2. in server.
 $ ./server
to run server.

3. in client,
 $ ./client <server-name> # server-name : IP address of server or name
to run client.

## if there are no probrems, the 1v1 chat program are run.
